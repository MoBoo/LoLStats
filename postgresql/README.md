# PostgreSQL Environment Variable Configuration
```
POSTGRES_DEFAULT_USER: <username> (default: airflow)
POSTGRES_DEFAULT_PASSWORD: <password> (default: airflow)
POSTGRES_DEFAULT_DATABASE: <database> (default: airflow)
POSTGRES_MULTIPLE_DATABASES: <username>:<password>:<database>[,<username>:<password>:<database>,...] (default="")
POSTGRES_IMPORT_DATABASES: <filepath>:<user>:<database>[,<filepath>:<user>:<database>,...] (default="")
```
